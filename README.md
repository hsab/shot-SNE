# shot-SNE
### Renarrativization of moving images based on visual similarity through the application of t-SNE and shot boundary detection

Made with openFrameworks.
Requires:
- [hecate](https://github.com/yahoo/hecate)
- ofxTSNE
- ofxOpenCv
- ofxCv
- ofxImGui
- ofxGui
- ofxJSON
- ofxPreset
- ofxXmlSettings
